# Messenger_Server ✈️

<br />
<br />

## 설치 방법

<br />

#### 1. npm 라이브러리 설치

<br />

```bash
npm install
```

<br />

#### 2. /config/default.json 변경
 
<br />

```json
{
    "mongoURI" : "mongodb+srv://<id>:<password>@<mongodb_server_ip>/<database_name>?retryWrites=true"
}
```

<br />

#### 3. 서버 실행

<br />

###### 기본포트 : 5000

```bash
npm run server 
```
